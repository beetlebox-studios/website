About

We build things
that move.

GraphX is a real-time graphics playground — a collection of shaders, simulations, and interactive experiments built on WebGL and WebGPU.

Each piece explores a different corner of the GPU pipeline: fluid dynamics, signed distance functions, particle systems, deferred rendering, and more. Built for the browser, no install required.

Open source. Contributions welcome.

Capabilities

  • Real-time Shaders GLSL / WGSL
  • Physics Simulations CPU & GPU
  • Procedural Generation Noise, SDF
  • Post-processing Multi-pass
  • Interactive Installations Web
